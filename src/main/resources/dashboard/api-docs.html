Personal Ops Hub

API Docs

Built-in reference for dashboard, automation, event, inventory, log, and management endpoints.

Auth Model

Dashboard and most API routes are protected by Basic Auth when OPS_ADMIN_PASSWORD is set. Mutating routes additionally require an authenticated dashboard session or OPS_ADMIN_TOKEN.

Keep public deployments behind HTTPS and configure OPS_ADMIN_PASSWORD before enabling management actions.

Read APIs

GET/api/health

App health, timestamp, version, and aggregate status.

GET/api/summary

Dashboard aggregate: current metrics, services, events, automation summary, and auth state.

GET/api/metrics/current

One current system metric snapshot.

GET/api/metrics/history?since=<epoch-ms>

Recent metric samples for chart rendering.

GET/api/services

Latest persisted service check results.

GET/api/events?limit=100&severity=WARNING&state=OPEN&q=disk

Recent events with optional severity, response state, and text filters.

GET/api/automation

Daily report, deploy, alert target, RSS, and page watch status summary.

GET/api/inventory

Read-only cron, systemd, Docker, and listening port inventory.

GET/api/logs/systemd?unit=<unit>&lines=100

Read-only journal logs for units allowed by OPS_ALLOWED_SYSTEMD_UNITS.

GET/api/logs/docker?container=<name>&lines=100

Read-only Docker logs for containers allowed by OPS_ALLOWED_DOCKER_CONTAINERS.

Mutating APIs

POST/api/services/run

Runs metric collection and configured service checks immediately.

POST/api/events/{id}/state

Updates an action-required event to OPEN, ACKNOWLEDGED, or RESOLVED.

POST/api/manage/actions

Runs allowlist-bound START, STOP, or RESTART for systemd units or Docker containers.

POST/api/alerts/test

Sends a test notification and records an INFO event.

POST/api/backups/report

Records backup success/failure from scripts or scheduled jobs.

Webhooks

POST/webhook/github

Verifies X-Hub-Signature-256 with OPS_GITHUB_WEBHOOK_SECRET and runs OPS_DEPLOY_COMMAND.

WebSocket

WS/ws/metrics

Streams current system metric snapshots every five seconds.

WS/ws

Compatibility endpoint. Use /ws/metrics for live dashboard data.
